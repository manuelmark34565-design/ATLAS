# User Flow

## Visitor
1. Visit homepage
2. Learn product benefits
3. Navigate to pricing or signup

## Registered User
1. Sign in / sign up
2. Access chat generator page
3. Start a new chat session
4. Ask questions and receive AI responses
5. Capture leads through conversation

## Business Owner
1. Sign up for ATLAS
2. Train the AI with business context
3. Invite team members (future)
4. Review captured leads and conversation history
